# HTML5 Intro Practice

This project contains my beginner HTML5 practice pages with CSS styling and media files.

## Topics Covered
- Basic HTML pages (headings, paragraphs, links, images)
- Forms
- Tables and layouts
- Flexbox practice (flex, center-flex, auto margin)
- Hero section and cards design
- HTML5 media (audio/video)

## How to Run
1. Download/clone the project
2. Open any .html file in a browser  
   Recommended start: hero.html or practice1.html
